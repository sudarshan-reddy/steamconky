Modded Steampunk conky with Hacker News Feed

(1) Install fonts in font folder. 

(2) Copy all other files to a folder in your home directory called ".conky"

(3) Use start_conky.sh to start up conkys. (You can set this to run on boot-up.)

(4) go get github.com/sudsred/getHackerNews and make sure bin is in your homepath
